# Annotation Resources



This directory contains external reference annotations required to reproduce the analysis. The files are publicly available from their original sources and should be downloaded before running the pipeline.



|**Subdirectory**|**Contents**|
|-|-|
|methylation/|Illumina HumanMethylation450 manifest and probe filtering resources|
|gene\_ontology/|Gene Ontology ontology file used for enrichment analysis|



&#x09;

#### Note



* Large annotation files are not tracked in the repository. Only the README files documenting their required sources are included.
