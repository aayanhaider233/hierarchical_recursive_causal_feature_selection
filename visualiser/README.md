# DAG Visualiser



This directory contains a self-contained interactive visualization of the inferred causal DAGs.



### Files

* **index.html:** visualization interface
* **graph\_data.js:** edge list encoded as source-target node pairs



Open **index.html** in a web browser to explore the ME-, gene-, and DMR-level graphs interactively. Each DAG is provided in a separate tab accessible through the interface.



The visualiser displays only nodes and directed edges; quantitative estimates (ACE and bootstrap edge frequencies) are provided separately in the results table.
